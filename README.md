# python-fundamentals-and-utility-tools
A collection of core Python utilities and Object-Oriented Programming (OOP) practice modules built to master data validation, file persistence, and programmatic logic

----
## Included Modules
### 1. ATM Simulator (`atm_simulator.py`)
* Object-Oriented Programming (OOP), Encapsulation, State Management
* Simulates core banking ATM operations using Python classes to handle deposits, balance checks, withdrawals securely and Account data


### 2. Task Manager / To-Do Application (`task_manager.py`)
* CRUD Logic (Create, Read, Update, Delete - Tasks), File I/O, Exception Handling, Mark & Categorise tasks
* CLI-based utility that allows users to manage operational tasks with persistent storage across sessions
* Save tasks to a file & load


### 3. Password Generator (`password_generator.py`)
* Randomization, User Criteria Inputs, String Manipulation
* Generates customizable, randomized passwords based on length, uppercase letters, numbers, and special symbols


### 4. Password Strength Checker (`password_checker.py`)
* Data Validation, Regular Expressions (Regex), Security Complexity Scoring
* Evaluates input passwords against standard security criteria and categorizes strength level with actionable recommendations
